# 🧠 Math Quiz Game

A modern and interactive Math Quiz Game built using HTML, CSS, and JavaScript. This project fetches real-time math questions from a free quiz API and provides a fun learning experience with multiple-choice questions, score tracking, and a clean responsive UI.

## 💻 Installation

Clone the repository:

```bash

git clone https://github.com/Bhola001/Modal-of-Quiz-game.git

```

Open the project folder:

```bash

cd quiz-game

```

Run the project by opening:

```bash

index.html

```

---

## 🎮 How to Play

1. Read the question carefully

2. Choose the correct option

3. Click Next Question

4. Check your final score after quiz completion
